
                CPP INHERITANCE
                ===============



Public inheritance
------------------
Class Base
class Derived: public Base

public members    of Base are publice members of Derived
protected members of Base are protected members of Derived





Protected inheritance
---------------------
Class Base
class Derived: protected Base

public members    of Base are protected members of Derived
protected members of Base are protected members of Derived





Private inheritance
-------------------
Class Base
class Derived: private Base

public members    of Base are private members of Derived
protected members of Base are private members of Derived
